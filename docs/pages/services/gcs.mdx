# Google Cloud Storage

## Installation

Install package

```sh
yarn add @uplo/service-gcs
```

Add service to Uplo

```ts
import GCSService from '@uplo/service-gcs'

const uplo = Uplo({
  services: {
    google: new GCSService({
      isPublic: false,
      credentialsPath: path.resolve(
        __dirname,
        './config/gcp-credentials.json'
      ),
      bucket: process.env.GCS_BUCKET,
    }),
  },
})
```

## Direct Upload

### CORS

This is example CORS configuration for direct uploads. Read how to configure
CORS on [Google Cloud Docs](https://cloud.google.com/storage/docs/configuring-cors).

```json
[
  {
    "origin": ["https://www.example.com"],
    "method": ["PUT"],
    "responseHeader": ["Origin", "Content-Type", "Content-MD5", "Content-Disposition"],
    "maxAgeSeconds": 3600
  }
]
```
