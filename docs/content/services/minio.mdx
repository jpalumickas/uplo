# MinIO

## Installation

Install S3 service

```sh npm2yarn
npm i @uplo/service-s3
```

Add service to Uplo

```ts
import S3Service from '@uplo/service-s3'

const uplo = Uplo({
  services: {
    minio: S3Service({
      isPublic: false,
      endpoint: 'https://minio.example.com',
      forcePathStyle: true,
      bucket: 'your-bucket',
      accessKeyId: '*****',
      secretAccessKey: '*****',
    }),
  },
})
```
