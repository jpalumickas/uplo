# Uplo Cloudflare Workers Example

Cloudflare Workers + Hono + Neon Postgres + Drizzle Adapter

## Requirements

* [Neon database](https://neon.tech)

## Usage

Create `.dev.vars` file:

```text
AWS_ENDPOINT=""
AWS_BUCKET=""
AWS_ACCESS_KEY_ID=""
AWS_SECRET_ACCESS_KEY=""
DATABASE_URL=""
UPLO_SECRET_TOKEN=""
```
